Telegram Bot Setup

Інструкція, як отримати токен бота та підключити вебхук.

  1. Відкрийте @BotFather у Telegram та виконайте команду /newbot.
  2. Вкажіть назву та унікальний username бота.
  3. Скопіюйте виданий HTTP API token.
  4. У файлі .env вашого серверу додайте змінну TELEGRAM_BOT_TOKEN=<token>.
  5. Перезапустіть сервер, щоб токен підтягнувся.
  6. Налаштуйте webhook: https://api.telegram.org/bot<token>/setWebhook?url=https://your-domain.com/webhook/<userId>
  7. Перевірте відповідь API Telegram, має повернути "ok": true.
Приклади команд для BotFather
  • /setdescription – додайте опис для користувачів.
  • /setuserpic – встановіть аватар.
  • /setcommands – опишіть доступні команди меню.

Після запуску бота адміністратор може ввести свій Telegram username в CRM налаштуваннях, після старту бота ID прив'яжеться автоматично.
